# Task 4: Data Visualisation 

## Overview 
Task 4 involved creating a visual representation of data to communicate insights clearly. 
The dashboard consolidates key business metrics into a single view to support data-driven decision-making. 

## Tools 
- Power BI 
- Power Query 

## Dashboard Features 
- Revenue and sales performance tracking 
- Customer and order overview 
- Product Performance analysis 
- Referral Source performance analysis 
- Monthly Revenue Trend analysis 
- Payment Method analysis 
- Order fulfilment analysis 
- Business insights and recommendations 

## Key Metrics 
- Total Revenue 
- Total Units Sold 
- Total Orders 
- Total Customers 
- Average Order Value
