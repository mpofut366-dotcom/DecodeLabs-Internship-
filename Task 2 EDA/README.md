## Overview and Objective

## Tools

## Key Findings 
- Average Order Value $1054, indicating relatively high transaction values 
- Quantity and Total Price showed a moderate positive correlation (0.62)
- Unit Price and Total Price showed the highest correlation (0.72)
- Instagram generated the highest order count, making it the most effective referral source
- Chairs recorded the highest quantity sold.
